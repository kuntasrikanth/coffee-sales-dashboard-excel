# ☕ Coffee Sales Dashboard — Excel

A full end-to-end Excel project built from raw data to interactive dashboard.

## What I built

Started with three messy tables — Orders, Customers, Products. 
Pulled everything together into one clean master sheet using XLOOKUP 
and INDEX MATCH, then built an interactive sales dashboard on top of it.

## What's inside

- XLOOKUP to merge customer & product data into orders
- INDEX MATCH for dynamic product detail retrieval
- IF functions to convert abbreviations → full names (e.g. ROB → Robusta)
- Duplicate removal & data cleaning
- Custom date formats, USD currency, metric size units
- Named Excel Table for auto-expanding pivot refresh
- 3 Pivot Tables → Line chart, Country bar chart, Top 5 Customers bar chart
- Timeline slicer + Roast Type / Size / Loyalty Card slicers
- All slicers connected to all charts simultaneously
- Clean dashboard UI — no gridlines, no headers, no formula bar

`Excel` `XLOOKUP` `INDEX MATCH` `Pivot Tables` `Data Cleaning` `Dashboard Design`
